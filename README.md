# ✝ 교회 PPT 자동화 도구

## 설치 방법

### 1. Python 패키지 설치
```
pip install flask python-pptx
```

### 2. 실행
```
python app.py
```

### 3. 브라우저에서 열기
```
http://localhost:5000
```

---

## 사용 방법

### ⚙️ 최초 설정 (설정 탭)
1. **찬송가 폴더**: 찬송가 PPTX 파일들이 있는 폴더 경로 입력
2. **기본 템플릿**: 매주 사용하는 템플릿 파일 경로 입력
3. **찬송가 슬롯**: 매주 고정적으로 찬송가를 넣는 위치 등록
   - 예: "1부찬양" → 슬라이드 3 이후, "2부찬양" → 슬라이드 8 이후
4. **설정 저장** 클릭

---

### 📋 주보 생성 (기능 1)

1. 템플릿 파일 경로 확인
2. 각 슬롯에 찬송가 번호 입력 (예: 23)
   - 자동으로 파일명에서 제목을 검색해 표시
3. 이번 주 사용 안 하는 슬롯은 "스킵" 체크
4. **PPT 생성하기** 클릭 → 파일 다운로드

---

### 🎵 성가대 가사 (기능 2)

1. 템플릿 파일 경로 입력 후 **슬라이드 보기**
2. 슬라이드 목록에서 클릭해서 선택:
   - 먼저 **제목 슬라이드** 클릭
   - 그 다음 **가사 템플릿 슬라이드** 클릭
3. 가사 입력 (빈 줄로 단락 구분)
4. **미리보기**로 슬라이드 분할 확인
5. **성가대 PPT 생성** 클릭

---

## 찬송가 파일명 규칙
```

100_나의갈길다가도록.pptx
```
- `번호_제목.pptx` 형식으로 저장
